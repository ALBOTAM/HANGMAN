# Hangman (Виселица)

Простая консольная игра "Виселица" на Python.  
Цель игры — угадать слово, вводя буквы по одной.

---

## Как играть

1. Склонируйте репозиторий или скачайте файлы:
```bash
git clone https://github.com/ALBOTAM/HANGMAN.git
cd HANGMAN
```
2. ```python hangman.py```
   
3. Вводите буквы, чтобы угадать слово.
   Если буква есть в слове — она откроется.
   Если буквы нет — количество попыток уменьшается.

4. Игра закончится, когда вы:
   
   Угадаете слово (победа), или
   Закончились попытки (проигрыш).

5.  Пример игры:
   
     Добро пожаловать в игру Виселица!
     слово: _ _ _ _ _
     Введите букву: a
     Правильно!
     Слово: a _ _ _ _
     ...

6. Требования:
   Python 3.x
   Запуск в терминале или командной строке
